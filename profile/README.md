## Products

### SlugBase

**Bookmark manager and link forwarding. Open source · Cloud available.**

The open-source stack is public on GitHub; **managed SlugBase Cloud is not in production yet**—we are finishing the hosted offering. **Self-host** with Docker (or your own infra) is the supported path while cloud launch approaches. Your bookmarks and short links stay under your control.

- [slugbase.app](https://slugbase.app)
- [docs.slugbase.app](https://docs.slugbase.app)
- [mdg-labs.dev/slugbase](https://mdg-labs.dev/slugbase)
- [github.com/mdg-labs/slugbase](https://github.com/mdg-labs/slugbase) (public)

### InboxOps

**Email support without the bloat.**

A shared inbox and helpdesk that connects to Gmail, Microsoft 365, or IMAP - built for teams that want clarity, not clutter. **Still in development**—not production-ready; closed beta when available.

- [inboxops.app](https://inboxops.app)
- [docs.inboxops.app](https://docs.inboxops.app)
- [mdg-labs.dev/inboxops](https://mdg-labs.dev/inboxops)
